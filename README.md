## Overview
CPC Valve Industry Web is a category driven website built to deliver a cleaner browsing experience than classic product-first navigation.

Instead of managing every page as a WooCommerce product, we use:

â€¢ WordPress categories as the main content backbone  
â€¢ A custom "category type" layer to shape different page templates  
â€¢ Custom PHP and custom HTML blocks for full control of UI and performance

This makes managing pages easier, keeps the navigation consistent, and improves the user journey.

## Why this architecture
Typical product-based setups become messy when you need:

â€¢ Many informational pages that are not real products  
â€¢ Multiple templates for different industrial categories  
â€¢ A fast and consistent UX across browsing, filters, and catalogue downloads

Our approach solves that by treating categories as first-class pages, then attaching a page behavior using category type.

## How it works
High level flow:

â€¢ Admin creates or updates Categories  
â€¢ Each category gets a Category Type (template behavior)  
â€¢ Frontend renders a category page with a custom HTML layout  
â€¢ Optional catalogue download is generated per category or per item

Common patterns used:

â€¢ Category landing page with featured cards  
â€¢ Subcategory grid and navigation  
â€¢ Template variations by category type  
â€¢ Catalogue download button mapped to a consistent URL pattern
